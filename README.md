# A-star
A* pathfinder algorithm in python with interface.

<i font-size>Just a little project for fun, messy code (updated after 2 years).</i>

# REQUIREMENTS
python 3.8+

Packages:
numpy\n
pygame

## CONTROLS
```
LMB = Choose node

ENTER = Go next step

R = Reset
```

## STEPS
```
1 - Choose start node

2 -  Draw a wall

3 - Choose end node

4 - Find a path
```
